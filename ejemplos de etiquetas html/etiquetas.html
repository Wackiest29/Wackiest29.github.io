Lista de Etiquetas HTML

<a>

La etiqueta <a> se utiliza para crear hipervínculos o enlaces.

Ejemplo de cómo usar:

                <a href="https://www.ejemplo.com">Visitar Ejemplo</a>

<p>

La etiqueta <p> se utiliza para crear párrafos de texto.

Ejemplo de cómo usar:

                <p>Este es un párrafo de ejemplo.</p>

<img>

La etiqueta <img> se utiliza para mostrar imágenes en una página web.

Ejemplo de cómo usar:

        <img src="imagen.jpg" alt="Descripción de la imagen">

<div>

La etiqueta <div> se utiliza para crear una división o contenedor genérico en una página web. Es comúnmente utilizada para agrupar y dar estructura a elementos HTML y aplicar estilos CSS a esos grupos de elementos.

Ejemplo de cómo usar:

        <div class="contenedor">
            <p>Este es un párrafo dentro del contenedor.</p>
            <ul>
                <li>Elemento 1</li>
                <li>Elemento 2</li>
            </ul>
        </div>

<ul>

La etiqueta <ul> se utiliza para crear listas desordenadas en HTML. Este tipo de lista muestra elementos en forma de viñetas o puntos.

Ejemplo de cómo usar:

        <ul>
            <li>Elemento 1</li>
            <li>Elemento 2</li>
            <li>Elemento 3</li>
        </ul>

<ol>

La etiqueta <ol> se utiliza para crear listas ordenadas en HTML. Este tipo de lista muestra elementos numerados o con viñetas ordenadas.

Ejemplo de cómo usar:

        <ol>
            <li>Primer elemento</li>
            <li>Segundo elemento</li>
            <li>Tercer elemento</li>
        </ol>

<li>

La etiqueta <li> se utiliza para representar elementos de lista tanto en listas ordenadas (<ol>) como en listas desordenadas (<ul>).

Ejemplo de cómo usar en una lista desordenada:

        <ul>
            <li>Elemento 1</li>
            <li>Elemento 2</li>
            <li>Elemento 3</li>
        </ul>

Ejemplo de cómo usar en una lista ordenada:

        <ol>
            <li>Primer elemento</li>
            <li>Segundo elemento</li>
            <li>Tercer elemento</li>
        </ol>

<table>

La etiqueta <table> se utiliza para crear tablas en HTML. Las tablas se utilizan para organizar y mostrar datos en filas y columnas.

Ejemplo de cómo usar:

        <table border="1">
            <tr>
                <th>Encabezado 1</th>
                <th>Encabezado 2</th>
            </tr>
            <tr>
                <td>Dato 1</td>
                <td>Dato 2</td>
            </tr>
            <tr>
                <td>Dato 3</td>
                <td>Dato 4</td>
            </tr>
        </table>

<tr>

La etiqueta <tr> se utiliza para definir filas en una tabla HTML creada con la etiqueta <table>.

Ejemplo de cómo usar:

        <table border="1">
            <tr>
                <td>Dato 1</td>
                <td>Dato 2</td>
            </tr>
        </table>

<td>

La etiqueta <td> se utiliza para definir celdas de datos en una tabla HTML creada con la etiqueta <table>.

Ejemplo de cómo usar:

        <table border="1">
            <tr>
                <td>Dato 1</td>
                <td>Dato 2</td>
            </tr>
        </table>

<th>

La etiqueta <th> se utiliza para definir celdas de encabezado de columna en una tabla HTML creada con la etiqueta <table>.

Ejemplo de cómo usar:

        <table border="1">
            <tr>
                <th>Encabezado 1</th>
                <th>Encabezado 2</th>
            </tr>
            <tr>
                <td>Dato 1</td>
                <td>Dato 2</td>
            </tr>
        </table>

<thead>

La etiqueta <thead> se utiliza para agrupar las filas de encabezado en una tabla HTML. Suele colocarse dentro de la etiqueta <table> y contiene uno o más elementos <tr> con celdas de encabezado <th>.

Ejemplo de cómo usar:

        <table border="1">
            <thead>
                <tr>
                    <th>Encabezado 1</th>
                    <th>Encabezado 2</th>
                </tr>
            </thead>
            <tr>
                <td>Dato 1</td>
                <td>Dato 2</td>
            </tr>
        </table>

<tbody>

La etiqueta <tbody> se utiliza para agrupar las filas de contenido en una tabla HTML. Suele colocarse dentro de la etiqueta <table> y contiene uno o más elementos <tr> con celdas de datos <td>.

Ejemplo de cómo usar:

        <table border="1">
            <thead>
                <tr>
                    <th>Encabezado 1</th>
                    <th>Encabezado 2</th>
                </tr>
            </thead>
            <tbody>
                <tr>
                    <td>Dato 1</td>
                    <td>Dato 2</td>
                </tr>
                <tr>
                    <td>Dato 3</td>
                    <td>Dato 4</td>
                </tr>
            </tbody>
        </table>

<tfoot>

La etiqueta <tfoot> se utiliza para agrupar las filas de pie de tabla en una tabla HTML. Suele colocarse dentro de la etiqueta <table> y contiene uno o más elementos <tr> con celdas de pie de tabla <td> o <th>.

Ejemplo de cómo usar:

        <table border="1">
            <thead>
                <tr>
                    <th>Encabezado 1</th>
                    <th>Encabezado 2</th>
                </tr>
            </thead>
            <tbody>
                <tr>
                    <td>Dato 1</td>
                    <td>Dato 2</td>
                </tr>
                <tr>
                    <td>Dato 3</td>
                    <td>Dato 4</td>
                </tr>
            </tbody>
            <tfoot>
                <tr>
                    <td colspan="2">Pie de tabla</td>
                </tr>
            </tfoot>
        </table>

<caption>

La etiqueta <caption> se utiliza para proporcionar un título o una descripción a una tabla HTML. Debe colocarse dentro de la etiqueta <table> y generalmente se muestra encima de la tabla.

Ejemplo de cómo usar:

        <table border="1">
            <caption>Ejemplo de tabla</caption>
            <tr>
                <th>Encabezado 1</th>
                <th>Encabezado 2</th>
            </tr>
            <tr>
                <td>Dato 1</td>
                <td>Dato 2</td>
            </tr>
        </table>

<form>

La etiqueta <form> se utiliza para crear formularios interactivos en HTML. Los formularios permiten a los usuarios ingresar datos que pueden enviarse al servidor para su procesamiento.

Ejemplo de cómo usar:

        <form action="procesar.php" method="post">
            <label for="nombre">Nombre:</label>
            <input type="text" id="nombre" name="nombre" required>
            <br>
            <label for="correo">Correo electrónico:</label>
            <input type="email" id="correo" name="correo" required>
            <br>
            <input type="submit" value="Enviar">
        </form>

<input>

La etiqueta <input> se utiliza para crear campos de entrada en un formulario HTML. Puede utilizarse para recopilar diferentes tipos de datos, como texto, contraseñas, opciones de selección, botones de radio y casillas de verificación.

Ejemplo de cómo usar:

        <label for="nombre">Nombre:</label>
        <input type="text" id="nombre" name="nombre" required>
        <br>
        <label for="contrasena">Contraseña:</label>
        <input type="password" id="contrasena" name="contrasena" required>
        <br>
        <label for="opcion">Opción:</label>
        <input type="radio" id="opcion1" name="opcion" value="opcion1">
        <label for="opcion1">Opción 1</label>
        <input type="radio" id="opcion2" name="opcion" value="opcion2">
        <label for="opcion2">Opción 2</label>
        <br>
        <label for="verificado">Verificado:</label>
        <input type="checkbox" id="verificado" name="verificado" value="verificado">

<textarea>

La etiqueta <textarea> se utiliza para crear un área de texto de múltiples líneas en un formulario HTML. Los usuarios pueden escribir texto largo en este campo, como comentarios o mensajes extensos.

Ejemplo de cómo usar:

        <label for="comentarios">Comentarios:</label>
        <textarea id="comentarios" name="comentarios" rows="4" cols="50"></textarea>

<select>

La etiqueta <select> se utiliza para crear un elemento de selección (menú desplegable) en un formulario HTML. Los usuarios pueden elegir una opción de una lista predefinida.

Ejemplo de cómo usar:

        <label for="frutas">Selecciona una fruta:</label>
        <select id="frutas" name="frutas">
            <option value="manzana">Manzana</option>
            <option value="banana">Banana</option>
            <option value="uva">Uva</option>
            <option value="naranja">Naranja</option>
        </select>

<button>

La etiqueta <button> se utiliza para crear un botón interactivo en un formulario HTML. Puede ser utilizado para realizar acciones específicas cuando el usuario hace clic en él.

Ejemplo de cómo usar:

        <button type="button" onclick="saludar()">Saludar</button>

        <script>
            function saludar() {
                alert('¡Hola, mundo!');
            }
        </script>

<label>

La etiqueta <label> se utiliza para etiquetar elementos de formulario en HTML. Proporciona una descripción o un nombre legible por humanos para un elemento de entrada, como un campo de texto o una casilla de verificación.

Ejemplo de cómo usar:

        <label for="nombre">Nombre:</label>
        <input type="text" id="nombre" name="nombre">
